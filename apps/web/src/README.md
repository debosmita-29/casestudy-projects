# Debosmita AI Portfolio

Deployable Vite + React + Tailwind portfolio app with the hero image included.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Vercel

- Framework Preset: Vite
- Build Command: `npm run build`
- Output Directory: `dist`
- Install Command: `npm install`
